 Kahoot Quiz App

A solo-player quiz application inspired by Kahoot, built with Django and vanilla JavaScript.

 Features

-  Leaderboard — compete against your own best scores
-  Per-question timer — answer before time runs out
-  Categories — choose your quiz topic
-  Clean and responsive UI

 Tech Stack

- **Backend:** Django (Python)
- **Frontend:** Vanilla JavaScript, HTML, CSS
- **Database:** SQLite

  Installation & Setup

1. Clone the repository
   git clone https://github.com/MirasNurzhanov/kahoot.git
   cd kahoot

2. Create a virtual environment
   python -m venv venv
   source venv/bin/activate

3. Install dependencies
   pip install -r requirements.txt

4. Run migrations
   python manage.py migrate

5. Start the server
   python manage.py runserver

6. Open your browser at http://127.0.0.1:8000
